Sprint 01: Construa um decodificador de texto com Javascript

Como funciona?

Quando o usuário digita o texto desejado e clica em "codificar", o texto é transformado em uma forma ilegível, utilizando algum tipo de método de codificação escolhido. Para visualizar o texto original, o usuário precisa colar o texto codificado no campo apropriado e clicar em "decodificar". O decodificador então reverte o processo de codificação, exibindo o texto original para o usuário.
